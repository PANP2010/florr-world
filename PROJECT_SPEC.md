# Florr World - 2D开放世界游戏引擎规格说明

> 版本：v0.1.0 | 状态：开发中

## 1. 项目概述

**Florr World** 是一款 2D 开放世界游戏，核心特色是"奇异物理"——在经典力学中加入量子隧穿、惯性奇点、时间常数偏移等奇特现象。

### 目标平台
- Python + Pygame（主引擎）
- 浏览器版（未来计划）

---

## 2. 核心系统

### 2.1 物理引擎（physics/）

#### 奇异物理层
- **惯性奇点**：物体在极端加速下惯性为零，产生瞬间位移
- **相位穿透**：物体可短暂进入"相位态"穿越障碍
- **重力异常层**：不同区域重力方向/强度不同
- **量子纠缠**：两个粒子状态即时关联，跨越任意距离
- **时间常数偏移**：局部区域时间流速变化

#### 物理模拟
- 碰撞检测（矩形/圆形/多边形）
- 刚体动力学
- 流体/粒子系统

### 2.2 玩家质点系统（player/）

- **属性**：质量、电荷、自旋、动能槽、连接端口
- **5种连接类型**：弹簧、铰链、融合、相位链接、引力连接
- **动能获取**：吸收环境能量、碰撞转换、能量转换

### 2.3 建造与工具系统（building/）

- **四纪元科技树**：
  - 第一纪元：基础工具
  - 第二纪元：机械装置
  - 第三纪元：能量科技
  - 第四纪元：量子科技
- **蓝图协议**：玩家可设计并保存结构蓝图
- **多人协作机制**（计划中）

### 2.4 核心玩法循环（core/）

1. **观察**：探索世界，发现异常现象
2. **假设**：提出解释，建立模型
3. **实验**：设计实验验证假设
4. **反馈**：根据结果修正认知
